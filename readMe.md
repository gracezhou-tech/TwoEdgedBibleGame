# TwoEdgedBibleGame
TwoEdged Bible Game: The fast-paced, shared-screen trivia challenge that tests your KJV Scripture knowledge against friends and family in 30-second rounds.

# first Version
By Grace Zhou
4 December 2025

# How to play the game
The app starts with a glowing Bible and a start button that plays a welcoming sound. You can set the number of players and their names on the setup screen.

The game flows through 7 questions, updating the scoreboard for each player's turn. When you reach the 7th question, a modal will pop up asking if you want to end or extend the game by another 7 questions. 
The end screen features a fanfare sound and an animated, rainbow-colored display for the winner, 
complete with a share button that generates the winning message formatted.
